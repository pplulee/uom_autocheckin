## 指令列表

```
/start - 开始使用
/ping - 测试是否在线
/help - 获取指令列表
/schools - 获取学校列表
/activities - 获取课程类型
/nextclass - 获取下一节课信息
/testlogin - 测试登录
/getlog - 获取日志

/fill <Unit Code> <课程类型> - 填写签到信息

例： 
/fill COMP12345 Lecture
/fill COMP66666 Lab
如课程类型不在列表中，直接填写即可，程序会自动选择Other并填写
```
