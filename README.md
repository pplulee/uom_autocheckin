## 指令列表

```
/ping - 测试是否在线
/schools - 获取学校列表
/activities - 获取课程类型
/testlogin - 测试登录
/cancel - 取消正在执行的任务

/fill <Unit Code> <课程类型> - 填写签到信息

例： 
/fill COMP12345 Lecture
/fill COMP66666 Lab
如课程类型不在列表中，直接填写即可，程序会自动选择Other并填写
```

**使用程序前请先确保设置了DUO默认推送设备**
